Quick Start (How to Run)
You don't need to touch a single line of code to use this.

1. Grab Your Executable
Go to the Releases section on the right side of this page and download the latest amanyu.ai.jar file.

2. Get a Free API Key (Takes 30 seconds)
Head over to Google AI Studio.

Sign in with any standard Gmail account.

Click Create API Key at the top left, copy the generated key string, and keep it safe.

3. Launch and Chat
Double-click the amanyu.ai.jar file to open the window.

Paste your key into the API KEY vault bar at the top.

Type your prompt at the bottom, hit Enter, and watch the engine spin up.
